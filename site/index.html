setup

Three steps. Then forget it.

No YAML. No plist files. No shell history full of tokens.

01 — token

Paste your Cloudflare API token

Burrow validates your token's scopes instantly. Required: Tunnel Edit, DNS Edit, Zone Read, Account Read. If a scope is missing, you see exactly which one.

02 — subdomain

Pick your zone and subdomain

Choose from a dropdown of your Cloudflare zones. Type a subdomain — Burrow provisions the CNAME m4.yourdomain.com and stores your credentials in Keychain.

03 — ssh

ssh from anywhere

Menu bar turns orange. Click "Copy SSH command." Paste in Terminal. Works from home, office, hotel WiFi, mobile hotspot — the hostname never changes.

capabilities

Built for reliability, not demos.

Burrow runs in the background, survives sleep/wake and network changes, and surfaces exactly what's wrong when something breaks.

Stable hostname

Named Cloudflare Tunnel = permanent UUID. Your ~/.ssh/known_hosts never breaks.

Keychain-only secrets

API token and tunnel run token live exclusively in macOS Keychain. Zero exposure in logs, plists, dotfiles, or shell history.

Sleep/wake recovery

Burrow observes sleep and wake notifications. Tunnel restarts within 10 seconds. Exponential backoff handles repeated failures gracefully.

Network change resilient

NWPathMonitor watches the default path. On WiFi switch or interface change, reconnect fires after a 2-second debounce. No manual intervention.

Terse, honest UI

"tunnel up. 6h 14m" — not "your secure connection was successfully established!" Missing scope shown verbatim with a copy button.

Near-zero overhead

Under 50 MB resident memory. Less than 0.1% idle CPU on Apple Silicon. The only process that matters is the cloudflared child.

requirements

What you need.

No subscription. No account with us. Just your existing Cloudflare setup.

  • macOS 13 Ventura+ Apple Silicon or Intel. Universal binary.
  • Cloudflare account Free tier. At least one zone managed in Cloudflare DNS.
  • cloudflared binary brew install cloudflared
  • API token scopes (4) Tunnel Edit, DNS Edit, Zone Read, Account Read. Burrow validates on paste.
  • SSH server enabled System Settings → Sharing → Remote Login. Burrow tunnels port 22 by default.
  • Outbound HTTPS/QUIC only cloudflared uses outbound-only connections. No inbound firewall rules needed.

install

Up in under five minutes.

No build tools. No npm. Drag, install, paste token.

1

Install cloudflared via Homebrew

brew install cloudflared
2

Download Burrow

Burrow.dmg

Developer ID signed · Notarized · macOS 13+

3

Drag to Applications, launch

Burrow appears in the menu bar. No Dock icon. The first-run wizard opens automatically — paste your Cloudflare API token and follow the prompts.

4

Create a Cloudflare API token

Visit dash.cloudflare.com/profile/api-tokens. Create a Custom Token with these four permissions:

Account → Cloudflare Tunnel → Edit Zone → DNS → Edit Zone → Zone → Read Account → Account Settings → Read
